Tugas: Laporan proses instalansi file UBUNTU 
![Screenshot 2025-02-06 193556](https://github.com/user-attachments/assets/3424ee90-99f8-40f7-951f-f74fe9d5181e)
1. Mengakses Situs Download Linux
	•	Gambar pertama menunjukkan halaman dari linux.org, tempat pengguna dapat memilih berbagai distribusi Linux.
	•	Pengguna memilih Ubuntu dengan mengklik “Download Ubuntu”, yang akan mengarahkan ke situs resmi Ubuntu.
![UBUNTU3](https://github.com/user-attachments/assets/97fb1cab-63f1-400b-9173-4ba7d6f83072)
2. Mengunduh Ubuntu Desktop
	•	Gambar kedua menunjukkan halaman download resmi Ubuntu (ubuntu.com/download).
	•	Pengguna memilih Ubuntu Desktop dan mengklik tombol hijau “Download Ubuntu Desktop” untuk mulai mengunduh file ISO.
![UBUNTU](https://github.com/user-attachments/assets/45791aa3-205c-406a-ab28-053dd4426f8c)
3. Proses Pengunduhan ISO Ubuntu
	- Gambar ketiga menunjukkan halaman konfirmasi setelah proses pengunduhan dimulai.
-	File yang diunduh adalah ubuntu-24.04.1-desktop-amd64.iso dengan ukuran sekitar 5.8 GB.
-	Pengguna dapat memantau proses download di browser.
